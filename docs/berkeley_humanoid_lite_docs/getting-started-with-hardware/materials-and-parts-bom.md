### PLA filament

The Berkeley Humanoid Lite robot is designed to be printed using standard PLA filament. Two recommended options are:

* [Bambu Lab PLA Basic Filament](https://us.store.bambulab.com/products/pla-basic-filament)
* [Hatchbox PLA Filament](https://www.amazon.com/dp/B00J0GMMP6)
